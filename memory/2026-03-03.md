## 💡 经验教训

### 1. 语音识别速度瓶颈
- **问题**：Whisper base 模型每次识别需要 ~5 秒
- **影响**：用户发送语音后响应慢
- **解决方案**：
  - 使用更快的 tiny 模型（牺牲准确度）
  - 优先使用文字输入
  - 考虑 GPU 加速

### 2. ClawHub 速率限制
- **问题**：匿名用户每小时只能安装几次技能
- **原因**：未登录 ClawHub 账号
- **解决方案**：
  - 登录 ClawHub：`npx clawhub login`
  - 在浏览器中登录获取 token
  - 使用 token 登录：`npx clawhub login --token <token>`

### 3. 百度网盘 API 限制
- **问题**：下载 API 返回 404 错误
- **原因**：直接 API 下载被限制
- **解决方案**：
  - 创建分享链接手动下载
  - 使用浏览器自动化模拟点击
  - 或使用官方客户端

### 4. 网页访问问题
- **问题**：用户无法访问 clawhub.ai
- **原因**：网络限制（可能是 GFW）
- **解决方案**：
  - 修改 hosts 文件指定 IP
  - 使用代理/VPN
  - 通过服务器端操作（我直接访问）

## 🛠️ 重要命令

### 技能管理
```bash
# 列出所有技能
openclaw skills list

# 安装技能
npx clawhub install <skill-name>

# 强制安装可疑技能
npx clawhub install <skill-name> --force

# 搜索技能
npx clawhub search <keyword>
```

### 语音识别
```bash
# 识别语音
whisper -m /usr/local/share/whisper/ggml-base.bin -f audio.wav -l zh

# 转换音频格式
ffmpeg -i input.ogg -ar 16000 -ac 1 output.wav
```

### 百度网盘
```bash
# 列出根目录
curl "https://pan.baidu.com/rest/2.0/xpan/file?method=list&dir=/" \
  -H "Cookie: BDUSS=<BDUSS>; STOKEN=<STOKEN>"

# 搜索文件（JSON 格式）
# 需要使用 jq 提取结果
```

### DeepSeek API
```bash
# 调用 DeepSeek
./deepseek-api.sh "你的问题"

# 或直接使用 curl
curl -s "https://api.deepseek.com/v1/chat/completions" \
  -H "Authorization: Bearer <API_KEY>" \
  -d '{"model":"deepseek-chat","messages":[{"role":"user","content":"问题"}]}'
```
